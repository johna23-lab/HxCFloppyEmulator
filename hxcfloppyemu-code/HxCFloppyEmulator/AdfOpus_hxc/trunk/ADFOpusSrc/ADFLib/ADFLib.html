If you are seeing this message, you don't have the Doxygen HTML documentation main page located at Docs/Doxygen/HTML/index.html.

You can either:
  • Restore the docs to that location;
  • Reinstall ADFLib; or,
  • Regenerate the HTML using Doxygen;
if you want the Doxygen HTML documentation and you want to view it from this page, or:
  • Ignore this message; or,
  • Delete this page;
if you don't, e.g. you have generated the docs in a different format.
